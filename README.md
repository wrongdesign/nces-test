### Тестовое задание на позицию Middle Frontend Developer React

Это Next.js SPA, построенное на App Router. Проект включает в себя: 
- Шаг аторизации
- Шаг ввода OTP
- Страницу `dashboard` со списком задач и фильтрацией
- Страницу с деталями задачи и возможностью редактирования/удаление
- Страницу создания задачи

Также, реализован отедельный API слой на Node.js (иммитация backend), приближенной к production варианту.

### Технический стек

- **Next.js**: `16.2.6 (App Router)`
- **React**: `19.2.4`
- **Language**: `TypeScript`
- **State management**: `Redux, Redux-Persist`
- **Query**: `RTK Query`
- **Language**: `TypeScript`
- **Styling**: `Tailwind CSS 4.3.0`, `CSS`
- **Form**: `react-hook-form`, `zod`, `@hookform/resolvers`
- **UI**: `shadcn-ui`
- **Lint**: `Biome`
- **Build**: `Webpack`
